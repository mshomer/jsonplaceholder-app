JSONPlaceholder

Fake Online REST API implementation
Powered by https://jsonplaceholder.typicode.com Server

Intro

JSONPlaceholder is a free online REST API that you can use whenever you need some fake data.
It's great for tutorials, testing new libraries, sharing code examples, ...

Resources

JSONPlaceholder comes with a set of 6 common resources in this app we only be using 3:

  • Users
  • Posts
  • Todos

Note: resources have relations. For example: user have many posts and todos

About this Application

This is an implementation of the JSONPlacholder server created with Angular framework,
which visualizes 3 pages: Users, Posts, Todos.
Each page fetch data from the fake server and display it to the user.
The user can do fake api request for: delete,update or create new item.
